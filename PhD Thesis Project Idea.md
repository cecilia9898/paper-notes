# 研究的大生物问题
- 持久化(persistence)到耐药性(resistance)转变是否遵循可预测的动力学规律？
- 能否通过系统建模预测并干预耐药性的出现？
- 细菌持久化状态下存在哪些动态生物学过程，哪些是导致不可逆耐药性出现的关键分子事件？

# 总体目标
系统定义并建模细菌从持久化（persistence）向耐药性（resistance）转变的动态生物学过程，并通过扰动实验验证关键调控因子的因果作用。

# Why is it important?
持久化细胞是治疗失败、慢性感染复发的根源之一，但从持久化到耐药性的分子轨迹没人系统研究过，搞清楚这一点是抗药性危机应对的重要突破口。

# How to answer it BEST?

- 我的问题不是：
“细菌是如何获得基因突变的？”（这个是Genome sequencing最适合回答的）
- 而是：
“细菌在耐药性出现之前，内部生理状态经历了什么功能性变化？”
- 不是去检测基因变化，而是去理解细胞功能状态的转变过程
- 而细胞功能状态，尤其是应激响应、代谢活性、DNA修复、膜运输系统等，最直接、最灵敏的反映载体是什么？
➔ 就是 **蛋白质表达水平和活性变化**

# 为什么 Proteomics 是最适合的方法？

| 层级 | 变化内容 | 最合适的技术 | 为什么 |
| :--- | :--- | :--- | :--- |
| 基因组 (DNA) | 是否发生了突变（如 gyrA、parC 突变） | Genome Sequencing | 检测点突变、耐药基因 |
| 转录组 (mRNA) | 某些基因是否转录上调/下调 | RNA-Seq | 捕捉转录活性变化 |
| 蛋白质组 (Proteins) | 细胞功能性反应（如代谢调整、应激响应、修复系统激活） | Proteomics | 蛋白质是执行功能的直接分子，调控和反馈都体现在蛋白层面 |

- 转录组和蛋白质组之间有显著偏差（不是所有mRNA变化都会反映在蛋白质水平上，尤其在应激和持久化状态下，转录后调控非常重要）
- 细胞功能的最终执行者是蛋白质（比如DNA修复酶、外排泵、氧化应激保护蛋白）
- 想要真正了解细胞生理功能状态如何随时间变化，并最终影响耐药性出现，Proteomics就是最合适的窗口
- 总结：我们研究的不是耐药性突变本身，而是使耐药性能够出现的动态功能重塑，而蛋白质组学是观察这一过程的最佳窗口。

# 逻辑
- “理解自然动态过程” ➔ “预测未来趋势” ➔ “探索干预手段”

# Specific Aims 
## Sub-Aim 1.1：时间分辨蛋白质组学刻画持久化到耐药性的动态转变

**思路：**
- **背景：**  
  持久化和耐药性是不同的生存策略，但中间的分子动态变化过程尚不清楚。

- **方法：**
  - 使用持续的抗生素压力（如 ciprofloxacin）刺激持久化群体。
  - 在多个关键时间点（0h, 6h, 12h, 24h, 48h）收集样本。
  - 采用TMT标记质谱或DIA质谱进行蛋白质组定量分析。

- **数据分析：**
  - 找出随时间变化的差异表达蛋白（DEPs）。
  - 进行功能富集分析（如GO/KEGG通路分析）。
  - 将蛋白按动态模式聚类，识别时间动态模块。
  - 建立非线性ODE模型，拟合蛋白质动态轨迹，寻找系统中的转折点（tipping point）。

- **预期结果：**  
  揭示持久化到耐药性转变过程中的主要动态生物学过程（如应激响应、DNA修复、代谢变化等），并形成可定量描述系统演变的动力学模型。


## Sub-Aim 1.2：基于扰动的关键调控因子因果验证

**思路：**
- **背景：**  
  时间分辨数据揭示了关联，但验证因果关系还需要主动干预实验。

- **方法：**
  - 根据1.1中识别出的关键动态调控蛋白（如应激反应sigma因子、DNA修复蛋白、外排泵调控因子等）优先筛选干预对象。
  - 使用基因编辑（如CRISPRi下调或可诱导过表达）或小分子抑制剂，对目标蛋白进行调控。

- **实验观察：**
  - 观察干预对细菌存活率、生长恢复速度、耐药性出现率的影响。
  - 进行干预后的小规模蛋白质组检测，了解通路响应变化。
  - 将干预前后细胞轨迹与ODE模型预测进行对比，判断干预是否能够延缓或阻断耐药性出现。

- **预期结果：**  
  验证哪些分子事件是真正驱动持久化到耐药性转变的关键因子，明确潜在干预靶点。

# 研究的大生物问题【胶质瘤】
- 少数细胞群体是如何通过功能性重编程动态逃避致死性压力的？
- 我们能否预测并干预它们从休眠状态向耐药性或复发性状态的转变？

# 生物学原理
- stress-induced cell fate transitions

# 新的【总体研究路线图】

| 阶段 | 具体应用 | 技术方法 | 目标 |
| :--- | :--- | :--- | :--- |
| 1. 细菌模型（短期） | 持久化 ➔ 耐药性 | 时间分辨 Proteomics + 动态系统建模 + 干预验证 | 建立技术和理论框架 |
| 2. 胶质瘤模型（中长期） | 休眠细胞 ➔ 复发肿瘤 | 时间分辨 Proteomics + 单细胞 Proteomics + 动态系统建模 + 干预验证 | 应用框架到人类肿瘤，发现复发预测标志物和干预靶点 |

- 一开始用细菌系统建立原理和工具；  
- 然后用胶质瘤系统做更高层次的人类疾病应用

# 总体目标
-系统定义并建模细胞在致死性压力下从休眠状态转变为耐药性或复发性状态的动态功能重编程过程，通过时间分辨蛋白质组学和动力系统建模揭示关键生物学过程和分子事件，并探索可预测和干预的节点

# Why is it important?
- 持久化（Persistence）和治疗后复发（Recurrence）是感染疾病和癌症领域中治疗失败的根本原因。
- 当前对抗生素耐药性和肿瘤复发的研究，往往聚焦于最终的基因突变事件，但忽略了在压力下支撑少数细胞存活的早期功能性适应过程。
- 系统地理解和预测细胞在应激压力下的功能轨迹变化，是在根源上设计新的干预策略、延缓或阻止耐药性和复发出现的关键。
- 在胶质瘤（glioma）中，如同在细菌感染中，休眠细胞在治疗压力下的存活与再激活，是导致几乎不可避免复发和治疗失败的主要原因。
- 通过针对这些早期动态功能适应过程进行干预，有望在疾病尚可逆转的阶段扭转命运。

# How to answer it BEST?
- 时间分辨定量蛋白质组学（Time-resolved quantitative proteomics）：直接观察蛋白质层面的动态功能变化，捕捉真实生理响应。
- 动力系统建模（非线性ODE建模）：对蛋白质动态变化进行建模，定量描述轨迹，识别系统转折点（tipping points），并模拟不同干预条件下的系统行为。
- 基于扰动的因果验证（Perturbation-based validation）：对识别出的关键调控因子进行基因或药物干预，验证其在持久化到耐药性/复发性转变中的因果作用。

这种综合策略不仅可以描述细胞的动态适应，还能够预测转变节点，并为早期干预提供理论依据。

# 为什么 Proteomics 是最适合的方法？
- 细胞在应激压力下的功能适应，包括代谢重塑、应激反应激活、DNA修复启动、膜转运调节等，最终都是通过蛋白质实现的。
- 基因组方法（如全基因组测序）能够检测突变，但无法反映实时的功能状态变化。
- 转录组方法（如RNA-Seq）可以检测mRNA水平变化，但mRNA和蛋白质之间的相关性在应激条件下通常较差，尤其存在大量转录后调控。
- 蛋白质组学可以直接测量细胞执行功能的分子，最真实地反映出细胞的功能状态。
- 此外，蛋白质动态变化往往早于稳定的基因变化，因此蛋白质组学为干预提供了更早、更可操作的时间窗口。

# 为什么一定要结合 Causal Inference？
1. 仅靠动态数据无法区分因果 vs. 相关
时间分辨蛋白质组学可以观察到哪些蛋白上升、下降，哪些通路活跃，但这些只是相关（correlation）。
仅靠时间变化，无法回答：
-是A蛋白引发了细胞从休眠转向耐药？
-还是两者共同受到了其他隐匿变量的影响？
- ✅ 因此，引入因果推断可以区分真正的driver（驱动因子）和bystander（旁观因子）

2. 没有因果推断，干预策略设计是盲目的
- 如果只是基于差异表达筛选蛋白，
➔ 你可能花很大精力干预某个上调的蛋白，结果发现干预后没有任何生物学效应，因为它只是一个响应标志物。
- 而如果有因果推断支持，
➔ 你能优先干预那些在动态演变过程中真正促成状态转变的关键调控因子。
- ✅ 这样你的CRISPRi knockdown、药物干预实验就更精准、更高效，节省大量试错时间和资源。

3. 生命系统具有复杂的反馈调控，不能简单用线性逻辑理解
特别是在持久化-耐药性转变或胶质瘤休眠-复发过程中，
存在多层级调控（比如代谢-信号传导-转录调控交织）；
存在正反馈、负反馈环路；
仅凭单变量动态变化，无法准确把握系统性驱动机制。
✅ 只有使用系统层面的causal discovery方法，才能重建这些复杂的调控网络。

4. 从“描述性科学”走向“预测性、干预性科学”的必经之路
传统生物学（尤其是Omics数据分析）：
描述：谁上调了，谁下调了。
现代生物学和精准医学要求的是：
预测：如果我在这个节点上干预，系统未来会走向哪条轨道？
控制：我能否通过合理干预，推迟或阻断耐药性或复发？
✅ 因果推断正是从观察性数据通向预测性、控制性科学的桥梁。

5. 对未来**跨物种应用（从细菌到胶质瘤）**具有高度通用性
在简单系统（细菌）上建立的因果推断框架， ➔ 可以直接迁移应用到复杂系统（人源胶质瘤休眠-复发模型）。
因为无论是简单系统还是复杂系统：
真正驱动状态转变的因子数量是稀疏的（sparse），
功能状态是通过少数关键节点调控的。
✅ 提前引入causal inference方法，能大大提升你未来处理复杂系统（比如肿瘤异质性）时的效率和准确性。
